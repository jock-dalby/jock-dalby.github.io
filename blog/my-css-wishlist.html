My CSS Design Wishlist!

Here is a list of design and structural features which I would like to add to my website. This is a living document and I will be continually adding more ideas, as well as crossing off items as they are completed.

  1. Improve the 'HTML, CSS & the DOM' blog page, by breaking it down into different columns and sections. Note: As the page has been designed mobile first, this may need to be applied as a media query.

  2. Add a drop-down menu for mobile screens and possibly larger screens too e.g. Uber

  3. Increase the size of the links in the nav so they fill the entire space of the button.

  4. Apply some transitions & transforms to add some 3D effects and/or animation to the page.

  5. Apply some graphics, imagery or iconography to the page USING CSS!!.

  6. Use opacity within a page to add fading effects and create depth using multiple layers.

  7. Use a vertical header.

  8. Use some pseudo-elements to style certain parts of an element or to add some generated content.

  9. Apply a left hand margin to the blog contents page (approx. col-3) and then a grid on the right with an image, a title and some sample text from the blog. Note: This should be applied as a media query as we are working mobile first.
